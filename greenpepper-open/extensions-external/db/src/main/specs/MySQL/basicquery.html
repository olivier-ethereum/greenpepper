Basic Query

With the DbFit system under test, Query fixture can be used to process a query and can be interpreted using any of the collections interpreters. The example below shows how to use it with the set of interpreter.


connect tomySQLdatabase onlocalhostwith usernamedbfit_userand passwordpasswordand databasedbfit

set ofqueryselect 'test' as x
x
test

Query can also work with multiple columns, and the order of columns does not matter

set ofqueryselect 1 as n, 'test' as x
xn
