Parameters as bind variables

Set interpreter can be used to set symbol values explicitly. They are then available to queries and statements as bind variables.


connect toOracledatabase onlocalhost:1521with usernamedftestand passworddftestand databaseXE

insert intoTableusers
usernamename
perapera pera
nujanuja nuja
nnnnnn nnn
setparameterp1pera

Symbols are then automatically available as bind variables for queries


set ofquerySelect * from users where username=:p1
usernamename
perapera pera
